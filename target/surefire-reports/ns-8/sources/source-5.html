36      * Checks if the matrix is positive definite. A matrix {@code M} is positive definite iff
37      * {@code x<sup>T</sup>Mx &gt 0} for any vector {@code x}, or equivalently, if the matrix is symmetric and
38      * all eigenvalues are strictly greater than zero.
39      *
40      * @param src Matrix to check if it is positive definite.
41      * @return True if the matrix is positive definite. Otherwise, returns false.
42      * @see #isPosSemiDef(Matrix) 
43      */
44     public boolean isPosDef(Matrix src) {
45         boolean result;
46         double tol = 1.0E-8; // Tolerance for considering eigenvalues positive.
47

49             result = Eigen.getEigenValues(src).min() > tol;

51             result = false;
