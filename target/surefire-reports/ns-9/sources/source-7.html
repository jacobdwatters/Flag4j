32  * <p>This class computes the Schur decomposition of a square matrix.
33  * That is, decompose a square matrix {@code A} into {@code A=QUQ<sup>H</sup>} where {@code Q} is a unitary
34  * matrix whose columns are the eigenvectors of {@code A} and {@code U} is an upper triangular matrix in
35  * Schur form whose diagonal entries are the eigenvalues of {@code A}, corresponding to the columns of {@code Q},
36  * repeated per their multiplicity.</p>
37  *
38  * <p>Note, even if a matrix has only real entries, both {@code Q} and {@code U} may contain complex values.</p>
39  */

103      * That is, decompose a square matrix {@code A} into {@code A=QUQ<sup>H</sup>} where {@code Q} is a unitary
104      * matrix whose columns are the eigenvectors of {@code A} and {@code U} is an upper triangular matrix in
105      * Schur form whose diagonal entries are the eigenvalues of {@code A}, corresponding to the columns of {@code Q},
106      * repeated per their multiplicity.</p>
107      *
108      * <p>Note, even if a matrix has only real entries, both {@code Q} and {@code U} may contain complex values.</p>
109      *
110      * @param src The source matrix to compute the Schur decomposition of.
111      * @return A reference to this decomposer.
112      */
113     @Override
114     public ComplexSchurDecomposition decompose(CMatrix src) {
115         ParameterChecks.assertSquare(src.shape);
116         hess.decompose(src); // Compute a Hessenburg matrix which is similar to src (i.e. has the same eigenvalues).
117         shiftedExplicitQR(hess.getH()); // Use the shifted QR algorithm.
118 
119         if(computeU) {
120             U = hess.Q.mult(U); // Convert Hessenburg eigenvectors to the eigenvectors of the source matrix.
121         }
122 
123         return this;
124     }
