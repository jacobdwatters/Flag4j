34 /**
35  * The base class for all tensors. A tensor is an algebraic object equivalent to a multidimensional array with a single
36  * data type.
37  * @param <T> Type of this tensor.
38  * @param <U> Dense Tensor type.
39  * @param <W> Complex Tensor type.
40  * @param <Z> Dense complex tensor type.
41  * @param <Y> Real Tensor type.
42  * @param <D> Type of the storage data structure for the tensor.
43  *           This common use case will be an array or list-like data structure.
44  * @param <X> The type of individual entry within the {@code D} data structure
45  */
